# Musafir_project

- Extract text from images (JPG/PNG)
- Extract text from videos/GIFs using frame extraction
- Preprocess images for high-accuracy OCR
- Extract structured entities such as:
- Name
- Date
- Amount
- PNR / Booking ID
- Flight route
- Passenger counts (adult male/female, minors)
- Handle Q&A queries about extracted content
- Parse flight booking queries into structured parameters
